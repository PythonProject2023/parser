Своя игра - интеллектуальная игра, в которой можно проверить свои знания, соревнуясь с другими игроками.
Структура игрового пакета:

- Пакет
   - Раунд 1
      - Тема 1
         - Вопрос 1
         - Вопрос 2
         - ...
      - Тема 2
         - ...
      - ...
   - Раунд 2
      - ...
   - ...

Правильный ответ на каждый вопрос приносит игроку некоторое количество очков, определяемое стоимостью вопроса.
Победителем становится игрок, набравший наибольшее количество очков.
